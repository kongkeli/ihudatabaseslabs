# University Database Labs 

This repository contains my laboratory exercises for the **Database Systems** course at the International Hellenic University (IHU).

It demonstrates basic database concepts using **Microsoft Access** to advanced SQL scripting with **SQL Server**.

## Repository Contents

* **Labs 01 - 04:** Database design and management using **Microsoft Access** (`.accdb` files).
* **Lab 05:** SQL Scripting using **SQL Server Management Studio (SSMS)**.
    * *Concepts covered:* Table Creation, Primary/Foreign Keys, Data Insertion (`INSERT INTO`), Data Types.

##  Technologies Used

* **Microsoft SQL Server 2022 / SSMS 20**
* **Microsoft Access**
* **T-SQL** (Transact-SQL)

## How to Run

### For SQL Files (Lab 05)
1.  Open **SQL Server Management Studio (SSMS)**.
2.  Connect to your local server instance.
3.  Open the `.sql` file.
4.  Execute the script to create the tables and insert the data.

### For Access Files
1.  Ensure you have **Microsoft Access** installed.
2.  Download the `.accdb` file and open it locally.
